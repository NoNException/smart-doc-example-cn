=== 测试https

*Type:* `GET`

*Author:* yu 2018/12/20.

*Content-Type:* `application/x-www-form-urlencoded;charset=utf-8`

*Description:* 测试https







*Request-example:*
----
curl -X GET -i http://localhost:8080/testHttps
----
*Response-fields:*

[width="100%",options="header"]

|====================
|Field | Type|Description|Since
|success|boolean|是否成功|-
