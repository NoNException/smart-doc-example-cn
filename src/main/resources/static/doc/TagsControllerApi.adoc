*Type:* `POST`

*Author:* yu 2020/7/2.

*Content-Type:* `application/json; charset=utf-8`

*Description:* 自定义mock tag获取mock值





*Request-parameters:*

[width="100%",options="header"]
[stripes=even]
|====================
|Parameter | Type|Description|Required|Since

|numbers|number|bigInteger|false|-
|====================

----
